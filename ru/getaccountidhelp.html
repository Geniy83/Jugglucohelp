Получить ID учетной записи

Как получить ID учетной записи Libreview, необходимый для сканирования датчика Freestyle Libre 3 с помощью NFC? Если вы указали "Вручную", вы можете ввести ID учетной записи вручную. Вы также можете отменить настройку “вручную” и нажать "Из Libreview", чтобы использовать адрес электронной почты вашей учетной записи и пароль для получения его из Libreview.

Также можно получить ID учетной записи Libreview из самого Libreview, а затем установить "вручную" и изменить адрес электронной почты и пароль учетной записи Libreview. Когда вы после этого установите "Отправить в Libreview" и нажмете "Повторно отправить данные", данные будут отправлены в эту другую учетную запись Libreview, но предыдущий ID учетной записи по-прежнему будет использоваться для сканирования датчика через NFC. Таким образом, Juggluco отправляет данные в учетную запись Libreview, отличную от той, которую использует приложение Abbotts Libre 3.
