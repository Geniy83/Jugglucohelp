К онфигурация Kerfstok

Исходный код Kerfstok можно загрузить по адресу https://www.juggluco.nl/Kerfstok/Kerfstok-source.html. Пользователи могут изменять его. Каждое приложение для часов имеет уникальный идентификатор. Если идентификатор отличается от идентификатора по умолчанию, Juggluco необходимо знать его для связи с приложением часов. Чтобы указать другой идентификатор, нажмите ID и введите шестнадцатеричную строку из 32 символов. Juggluco на одном устройстве можно подключить только к одному приложению на одних часах. Изменения приведут к потере данных.

Ярлыки

Ярлыки отправляются в приложение Kerfstok для часов Garmin, чтобы можно было легко вставлять определенные значения.
