О программе

Juggluco написал Jaap Korthals Altes

Помимо стандартных библиотек Android, используются следующие:

  • Команды NFC и Bluetooth, специфичные для датчиков Abbott FreeStyle Libre, и алгоритмы интерпретации, специфичные для датчиков FreeStyle Libre. Все они не документированы

  • Для связи с часами Garmin используются Connect IQ Mobile SDK;

  • Графики создаются с помощью NanoVG;

  • Функция зеркального отображения использует Ascon authenticated encryption;

  • Связь с датчиками Libre3 использует TinyCrypt;

  • Средство выбора цвета представляет собой адаптацию AmbilWarna;

  • Поддержка NFC NovoPens - это модификация кода, взятого из xDrip.
