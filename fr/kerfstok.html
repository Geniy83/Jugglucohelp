Kerfstok

Cette vue est utilisée pour résoudre les problèmes de connexion avec Kerfstok ( https://apps.garmin.com/en-US/apps/b6348ccc-86d8-4780-8013-d9e19fed5260 (voir aussi https://www.juggluco.nl/Kerfstok /index.html ).


Kerfstok est une application de surveillance pour les smartwatches Garmin programmables avec écran tactile. Kerfstok a les utilisations suivantes :

  • Entrez les quantités et échangez-les avec Juggluco;

  • Affiche la valeur de glucose du flux reçu de Juggluco ;

  • Affiche cette valeur de glucose pendant le sport, tout en indiquant également la vitesse et la distance. L'activité sera enregistrée et pourra être affichée dans l'application Connect de Garmin.

Cette application de montre peut se connecter via Bluetooth avec Juggluco pour échanger des quantités et afficher en permanence votre valeur de glucose actuelle.


Lorsqu'un problème survient, cet écran peut aider à en trouver la cause :


Tout d'abord, le nom de votre montre doit être affiché " CONNECTÉ ". Si ce n'est pas le cas, résolvez le problème dans l'application Garmin Connect. Deuxièmement, Kerfstok doit être démarré sur votre montre.

Parfois, la connexion entre la montre et le téléphone est interrompue.


Si l'état d'envoi est FAILURE_DURING_TRANSFER, vous pouvez effectuer une ou toutes les actions suivantes :

- Éteignez et rallumez Bluetooth, puis appuyez sur Reinit ;

- Redémarrez la montre (cela fonctionne presque toujours, mais parfois il faut le faire plusieurs fois) ;

- Redémarrez le téléphone (dernière option).


Pour d'autres problèmes, forcez l'arrêt de l'application Garmin connect, redémarrez-la et appuyer sur réinitialiser peut être utile.


S'il y a une bonne connexion et que, pour une raison quelconque, les quantités sur la montre et le téléphone ne sont pas synchronisés, vous pouvez appuyer sur Sync pour qu'ils s'envoient les valeurs qui n'ont pas encore été envoyées.


Au démarrage de Juggluco, Sync est automatiquement activé. Cela signifie qu'après avoir réinstallé Kerfstok, toutes les quantités seront également supprimées dans Juggluco. Le seul moyen de conserver ces quantités, c'est de créer un miroir et de le renvoyer à Juggluco. Soyez prudent car le miroir est également immédiatement synchronisé avec son original. Pour fonctionner comme une sauvegarde, toute connexion avec l'original doit être rompue à un moment donné. Vous pouvez également connecter la montre à un appareil qui est un miroir (menu central gauche->Miroir) de l'appareil dans lequel vous avez entré les quantités.


Actualiser recharge cette boîte de dialogue avec de nouvelles informations (par exemple, après Reinit ou Sync).


Cocher " Glucose " détermine si les valeurs de glucose sont envoyées à la montre.


Parfois, un bouton nommé « Envoyer la file d'attente » s'affiche ; cela signifie qu'il y a des messages qui ne sont pas encore envoyés à la montre. Normalement, ils seront envoyés automatiquement en temps voulu, mais parfois cela a été interrompu et appuyer sur la file d'attente d'envoi poursuivra l'envoi. Mais s'il n'a pas été envoyé à cause d'une connexion rompue, cela a très peu d'avantages. Appuyer sur Envoyer la file d'attente pendant une transaction en cours peut interrompre la connexion.
